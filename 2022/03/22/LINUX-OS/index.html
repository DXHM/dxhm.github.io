Linux系统基础

  1. MBR：主引导记录
  2. grub2引导
    (grub「grand unified bootloader 宏大的统一引导启动程序」：幼虫，搜寻)
    1. boot.img
    2. core.img
    3. *.mod
    4. grub.cfg
      a. 未独立/boot：/boot/grub2/grub.cfg
      b. 独立/boot:/grub2/grub.cfg
    5. grun2-setpassword：设置引导密码
  3. init：PID=1（进程号）
    1. centos7第一个进程：systemd（配置文件：/usr/system/systemd/）
    2. 执行登录 /bin/login
  4. demesg | less：“|”管道符，左侧命令输出作为右侧的输入
  5. /dev：设备 –> 文件
  6. /usr：unix shared resources（共享资源）「类似win的Program Files」
  7. /src：原文件
  8. /tmp：临时文件
  9. /var（variable）：日志文件、打印机脱机工作文件、缓冲区
  10. /opt（option）：选项文件
  11. /mnt（mount）：挂载
  12. cd：change directory
  13. 文件类型：
    1. d：目录
    2. l：链接
    3. b：块设备
    4. c：字符设备文件
    5. -：普通文件
  14. 文件权限：
    1. 用户组：U / G / A
    2. 九位字母表示法：d[文件类型] rwx[文件属主] r-x[同组用户] r–[其他用户]
  15. cat快速编辑：
    1. cat>test
    2. [输入字符]
  16. tail & head：查看后十行，前十行 //-n num : 指定行数
  17. mdir [ -m ] [ -p 目录名 ]
  18. rmdir（remove dir）：用于删除空目录
    1. rm -r
  19. touch：修改时间属性 * （不存在）创建文件
  20. 软链接：ln -s
  21. 硬链接：ln
    1. 以文件副本形式存在，不占用实际控件
    2. 不允许目录硬链接
    3. 只有在同一个文件系统才允许硬链接
  22. 查找类：which、whereis、find、grep
  23. 文件重定向：
    • /dev/stdin：标准输入文件
    • /dev/stdout：标准输出文件
    1. 输出重定向
      1. “>”：文件存在，覆盖
      2. “>>”：文件存在，添加内容到文件末尾
    2. 输入重定向
      1. “<”
    3. 管道符重定向：“|” 左输出为右输入
